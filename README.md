一款基于Web的经典2048游戏，具有丰富的功能和精美的界面设计。

## 📁 项目结构

```
.
├── index.html          # 游戏主页面
├── 2048-styles.css     # 样式文件
├── 2048-game.js        # 游戏逻辑
```

## ✨ 功能特性

### 游戏核心功能
- 🎯 经典2048游戏玩法
- ⌨️ 支持键盘方向键控制（↑↓←→ 或 WASD）
- 📱 支持触屏滑动操作
- 💾 自动保存最高分记录

### 排行榜系统
- 🏆 记录玩家昵称和分数
- 📊 支持按分数/时间排序
- 🗑️ 一键清空排行榜记录

- 🎯 **首次通关** - 获得2048方块
- ⭐ **满分挑战** - 获得10000分
- ⚡ **速度之王** - 在1分钟内获得2048
- 🎖 **坚持就是胜利** - 连续玩10局

### 分享功能
- 📤 生成分享文本
- 📋 一键复制到剪贴板

### 个人简历页面
- 📄 从游戏界面可跳转查看
- 👤 包含个人信息、教育背景、项目经验等

### 视觉体验
- 🎨 精美渐变背景设计
- ⏳ 游戏加载动画
- 🎮 首次进入教程引导
- 📱 响应式布局，支持移动端

## 🎮 游戏说明

### 操作方式
- **键盘**: 使用方向键 ↑↓←→ 或 WASD 移动方块
- **触屏**: 在屏幕上滑动手指移动方块

### 游戏规则
1. 相同数字的方块碰撞时会合并成它们的和
2. 每次移动后会随机生成一个新方块（2或4）
3. 无法移动时游戏结束
4. 达到2048即为胜利

## 📝 开发说明

### 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 游戏主页面，包含所有弹窗和UI结构 |
| `2048-styles.css` | 完整的样式文件，包含响应式设计 |
| `2048-game.js` | 游戏核心逻辑，包含排行榜、成就等功能 |

### 数据存储

游戏使用 `localStorage` 存储以下数据：
- `bestScore` - 最高分
- `leaderboard` - 排行榜数据（JSON格式）
- `achievement_*` - 成就解锁状态
- `showTutorial` - 是否显示教程
- `gameStreak` - 连续游戏次数
