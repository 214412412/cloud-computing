# 2048 终极版

一款基于经典2048游戏开发的增强版网页游戏，添加了丰富的功能和精美的界面设计。

## 🎮 游戏特性

### 核心玩法
- 经典2048游戏玩法，通过滑动合并方块
- 支持键盘方向键和触屏滑动操作
- 响应式设计，适配各种屏幕尺寸

### 新增功能
- **个人简历页面**：从游戏界面顶部导航可跳转查看开发者简历
- **排行榜系统**：支持记录玩家昵称、按分数/时间排序、可清空记录
- **成就系统**：4个精心设计的成就等你解锁
  - 🎯 首次通关：获得2048方块
  - ⭐ 满分挑战：获得10000分
  - ⚡ 速度之王：在1分钟内获得2048
  - 🎖 坚持就是胜利：连续玩10局
- **游戏分享**：一键复制成绩到剪贴板，与好友分享战绩
- **加载动画**：游戏启动时有精美的加载画面
- **操作教程**：首次进入游戏显示操作说明，可选择不再提示

## 🎯 游戏操作

### 键盘操作
- `↑` / `W` - 向上移动
- `↓` / `S` - 向下移动
- `←` / `A` - 向左移动
- `→` / `D` - 向右移动

### 触屏操作
- 在屏幕上滑动对应方向即可移动方块
